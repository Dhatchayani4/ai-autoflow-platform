# AutoAI Flow

AutoAI Flow is a modern AI automation landing page built using React, Vite, and Tailwind CSS.

## Features

- Modern responsive UI
- Interactive Hero section
- AI Features section
- Workflow showcase
- Dashboard preview
- Pricing cards with currency switch
- Testimonials carousel
- FAQ accordion
- Call-to-Action section
- Smooth scrolling navigation
- Responsive design
- Custom animations

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## Getting Started

Clone the repository

```bash
git clone https://github.com/Dhatchayani4/ai-automation-platform.git
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build the project

```bash
npm run build
```
